Break Point

font-family

font-familyのmixinです。フォントはプロジェクトごとに指定します。使用しないコードはそのままにせず削除ください。

汎用class
class 用途
@include font-webfont_noto_sans; Noto Sans JP GoogleFontsでよく使う日本語ゴシック体
@include font-webfont_noto_serif; Noto Serif JP GoogleFontsでよく使う日本語明朝体
@include font-JP_sans; 各デバイスの日本語デフォルトゴシック体
@include font-JP_serif; 各デバイスの日本語デフォルト明朝体
@include font-EN_sans; 各デバイスの英語デフォルトゴシック体
@include font-EN_serif; 各デバイスの英語デフォルト明朝体
@include font-EN_serif; 各デバイスの英語デフォルト明朝体
@include font-heading; 見出し用font-family
@include font-code; プログラムコード表示用font-family

See the Pen font-family by harumi-sato (@harumi-sato) on CodePen.

まとめ

汎用classのマークアップとclass名一覧です。厳守するものではなくプロジェクトにより柔軟に対応する。
コピペしてそのまま使うのではなく、サイトとしての用途とコーディング後の運用や実装の効率性を考えて設計する。

汎用class
class 用途
div.l-container body直下のレイアウト用div
header.l-header ヘッダー
main.l-main メインカラム
article.l-article メインカラム
aside.l-sub サブカラム
footer.l-footer フッター
